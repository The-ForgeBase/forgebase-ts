# Package 2 Documentation

Welcome to the documentation for Package 2. This guide will provide you with all the necessary information to get started and effectively use Package 2 in your projects.

## Installation

To install Package 2, use the following command:

```bash
npm install package2
```

## Usage

Here are some examples of how to use Package 2 in your code:

### Example 1: Basic Usage

```typescript
import { Package2 } from 'package2';

const instance = new Package2();
instance.doSomething();
```

### Example 2: Advanced Usage

```typescript
import { Package2 } from 'package2';

const instance = new Package2({ option: 'value' });
instance.doSomethingAdvanced();
```

## API Reference

### `Package2`

#### Methods

- `doSomething()`: Performs a basic operation.
- `doSomethingAdvanced(options: object)`: Performs an advanced operation with the given options.
