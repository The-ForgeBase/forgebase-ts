# Package 3 Documentation

Welcome to the documentation for Package 3. This guide will provide you with all the necessary information to get started and effectively use Package 3 in your projects.

## Installation

To install Package 3, use the following command:

```bash
npm install package3
```

## Usage

Here are some examples of how to use Package 3 in your code:

### Example 1: Basic Usage

```typescript
import { Package3 } from 'package3';

const instance = new Package3();
instance.doSomething();
```

### Example 2: Advanced Usage

```typescript
import { Package3 } from 'package3';

const instance = new Package3({ option: 'value' });
instance.doSomethingAdvanced();
```

## API Reference

### `Package3`

The main class provided by Package 3.

#### Methods

- `doSomething()`: Performs a basic operation.
- `doSomethingAdvanced(options: object)`: Performs an advanced operation with the given options.
